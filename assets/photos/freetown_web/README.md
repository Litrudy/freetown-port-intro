# 弗里敦港 · 网页素材包

仿沉浸式作品集风格(参照 lusion.co:深色、电影感、左图右文、滚动换图)的弗里敦港介绍网页,
配套一份**自由版权**的弗里敦/塞拉利昂图片分类清单。

## 目录结构
```
freetown_web/
├─ index.html              # 网页(双击即可在浏览器打开)
├─ README.md               # 本说明
└─ images/
   ├─ manifest.csv         # 图片清单(可用 Excel 打开)
   └─ manifest.md          # 图片清单(可读版,含分类与来源)
```

## 关于图片(重要)
- 由于生成环境的网络限制,**无法把图片原始文件下载并打包进本包**;
  网页里的图片是通过 Wikimedia Commons 的**公开直链在你的浏览器里实时加载**的——
  你本地双击 `index.html` 打开时,联网即可正常显示。
- 这些图片来自 **Wikimedia Commons(自由版权,多为 CC BY / CC BY-SA / 公有领域)**。
- **若用于正式/商用发布,请:**
  1. 打开 `images/manifest.md` 中每张图的 **Commons 文件页**,核对其**具体授权条款**;
  2. 按要求**署名**(作者 + 授权链接 + 是否修改);
  3. **下载原图后自托管**(放到自己的服务器/CDN),不要长期热链 Wikimedia。

## 如何替换 / 增删图片
打开 `index.html`,在 `<script>` 顶部的 `IMAGES` 数组里编辑即可:
```js
const IMAGES = [
  { url:"图片直链或本地路径,如 images/port.jpg", grad:"加载失败时的渐变兜底" },
  ...
];
```
每个右侧文字段落 `<section>` 上的 `data-img="序号"` 决定它对应左侧第几张图,
`data-cap` 是左下角图注,`data-idx` 是右侧竖排索引。

## 自托管做法(推荐)
1. 从清单里各 Commons 文件页下载原图,改名后放进 `images/`(如 `images/port.jpg`);
2. 把 `IMAGES` 里的 `url` 改成 `"images/port.jpg"`;
3. 整个文件夹上传到你的静态空间即可。

## 设计说明
- 深色背景 + 蓝绿双色调滤镜 + 颗粒噪点,贴近 lusion 的冷峻电影感;
- 左侧大图固定,随右侧滚动**交叉淡入切换**,并带轻微 Ken-Burns 缩放;
- 顶部进度条、竖排索引、滚动提示、文字逐段浮现;
- 移动端自动切换为上图下文的纵向布局;
- 图片加载失败时**自动回退为同色系渐变**,版面不会塌。

> 文案取自《弗里敦港·物流实务介绍》。数据(水深、费率、引航等)可能随港方调整变动,以代理与港务局最新确认为准。
